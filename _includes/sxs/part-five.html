Part five Add the USWDS Banner

Get the USWDS Banner HTML.

Head over to the USWDS Banner component page. Expand the Component code accordion and copy the first section of markup, from <section class="usa-banner"...> to </section>

Add the Banner HTML into the tutorial markup.

Paste the Banner markup into the top of the tutorial index file (index.md), right after the front matter.

description index.md

---
layout: default
title: USWDS Tutorial
---

<section class="usa-banner" aria-label="Official government website">
  <div class="usa-accordion">
    <header class="usa-banner__header">

    ...

    </header">
  </div>
</section>
<header class="opener">

Update image paths.

Now the banner is on your page. But when you look at it in the browser window, it won't look right. The links are broken. That's because the image path in our tutorial is different from the image path on the USWDS website. Let's update all the image paths to our project path.

In index.md, search for all instances of the broken path (/assets/img) and replace each one with the our project's USWDS image path (/assets/uswds/img).

The example shows each of these replacements. Use find/replace to replace all of them.

description index.md

src="/assets/img/us_flag_small.png"
src="/assets/uswds/img/us_flag_small.png"


src="/assets/img/icon-dot-gov.svg"
src="/assets/uswds/img/icon-dot-gov.svg"


src="/assets/img/icon-https.svg"
src="/assets/uswds/img/icon-https.svg"

Once you're updated all these image paths, your banner should look pretty good!
